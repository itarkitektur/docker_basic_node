# Run

Run all containers:

```bash
docker-compose up --build
```

Stop containers: 

```bash
docker-compose down
```

# Infrastructure

### Initialize (First time only)

```bash
$ terraform init
```

### Create Infrastructure

1. Validate the configuration:

```bash
$ terraform 
```

2. Create the infrastructure:

```bash
$ terraform apply
```

### Destroy Infrastructure

```bash
$ terraform destroy
```
